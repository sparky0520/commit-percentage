# Commit Percentage

This is a vs code extension that will tell you the percentage changes (deletion and additons are the same here) of your staged files compared to the total repository lines of code.

## Installation

- `npm install`
- `npm run compile`

## Debugging

- Press F5 in the vs code window
- Open a repository in the newly opened window
- Make changes and stage them using '+' icon.
- Check the bottom left status bar.
